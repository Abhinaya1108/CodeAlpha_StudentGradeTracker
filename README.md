# Student Grade Tracker using Java

This project was developed as part of the CodeAlpha Java Programming Internship.

## Project Description
The Student Grade Tracker is a Java console-based application that allows users to input student names and grades, store them using ArrayList, and generate a summary report.

## Features
- Add student names and grades
- Store student data using ArrayList
- Calculate average score
- Identify highest score
- Identify lowest score
- Display summary report

## Technologies Used
- Java
- Object-Oriented Programming
- ArrayList
- Scanner

## How to Run
1. Compile the program:
javac Student.java StudentGradeTracker.java

2. Run the program:
java StudentGradeTracker
